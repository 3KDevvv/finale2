About CardFlux

CardFlux is a Magic: The Gathering deck builder that lets you search cards, save favorites, and build custom decks using a live MTG Database

Features

Search Cards

Search cards you like and view them anytime.

Favorites

Save cards you like and view them anytime.

Deck Builder

Create decks, add cards, and orgaanize your collection..

Deck Stats

See card totals, creatures, and mana cost breakdowns.

How to Use

  • Search cards from the Search page
  • Add favorites using the + button
  • Create up to 3 decks
  • Each deck holds up to 20 cards!
